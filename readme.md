## Projektet används vid genomgångar på Javascript

### För att klona projektet

1. Skapa en tom katalog och öppna den med VS Code
2. Öppna en terminal och skriv in (*glöm ej punten på slutet*):

```bash
git clone git@github.com:ByHe/te2inf-js-2026.git .
```
### För att uppdatera projektet
Skriv i terminalen

```bash
git pull
```
